## Mateus Mendes de Freitas

My name is Mateus, I am 20 years old, and I started my studies at IFPR Ponta Grossa in 2022. Currently,I have completed a Technical Course in Information Technology.

I have experience with relational databases such as MariaDB and MySQL. Regarding programming languages, I have knowledge of JavaScript, PHP, Python, and Java.

With the Laravel framework, I learned how to use PHP for full-stack development based on the MVC (Model-View-Controller) architecture. With Django, I used Python for the same purpose, applying the MVT (Model-View-Template) architecture.

Additionally, I studied React with the goal of learning how to integrate modern interfaces with the backend frameworks I already know.

Recently, I acquired knowledge about Docker to better understand how Laravel Sail works.
